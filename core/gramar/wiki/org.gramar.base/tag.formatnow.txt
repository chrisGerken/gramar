
==== formatNow Tag ====

The formatNow tag writes the current time using the format specified.


^ Attribute      ^ Kind   ^ Required    ^ Purpose          ^
|**pattern**  |string  |yes  |The format of the date to be written.  See SimpleDateFormat for more specifics.  |

=== Full Form ===

<**c:formatNow**  **pattern**="" >
